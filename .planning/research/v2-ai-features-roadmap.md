## 1. North-Star Vision

Hypespace is the "AI event team in a box" for local businesses that don't have marketing staff. A dentist, realtor, or small business owner should be able to go from "I want to host an event" to "event is planned, promoted, attended, and followed up on" in under 30 minutes, with AI doing 80% of the work.

**Core promise to the customer:** *Describe your event in one sentence. We'll plan it, promote it, fill the room, and follow up with every attendee.*

## 4. Industry-Specific Playbooks

Each vertical gets a pre-built "playbook" — bundled templates + AI presets that turn Hypespace from general-purpose into *obviously-for-you*.

### 4.1 Dentist Playbook

- Pre-loaded event types: new patient open house, whitening day, Invisalign event, CE evening, community kids' day, dental emergency seminar.
- HIPAA-safe copy templates (no PHI in marketing).
- Referral loops: AI identifies top-referring patients, drafts "bring a friend" campaigns.
- Reactivation: AI pulls inactive patients (via PMS integration: Dentrix, Open Dental, Eaglesoft) and builds a comeback campaign.
- Insurance / benefits reminder event ("Use it before you lose it — December event").
- Post-event: auto-book consults, send after-visit instructions, request Google reviews.

### 4.2 Healthcare Practice Playbook

- Pre-loaded: Botox / filler launch nights, med spa grand opening, patient education webinar, chronic disease community event, clinical trial recruitment.
- Compliance rails: FDA + state-specific claim guardrails, disclaimers auto-inserted.
- Referring-physician events: AI drafts CME-style invitations and follow-up.
- Procedure-interest scoring: AI scores attendees by likelihood to book; routes to front desk.
- Integration priority: Jane, DrChrono, athenahealth, Tebra.

### 4.3 Realtor Playbook

- Pre-loaded: open house, broker opens, buyer seminar, investor meetup, client appreciation BBQ, past-client referral brunch.
- Property-aware: pulls listing info from MLS / Zillow / Realtor.com; generates flyer, social kit, landing page per listing in minutes.
- Neighborhood intelligence: AI adds local data (schools, crime, trends) to the marketing.
- Open house lead capture + auto-nurture (30-day drip).
- Referral farming: quarterly client-appreciation events on autopilot.
- Integration priority: Follow Up Boss, kvCORE, Sierra Interactive, LionDesk, Moxi.

### 4.4 SMB Playbook

- Pre-loaded: grand opening, customer appreciation, workshop, product launch, networking mixer, holiday event.
- Google Business Profile sync: event auto-published.
- Yelp / Nextdoor / Eventbrite cross-posting.
- POS / loyalty integration (Square, Toast, Shopify) → invites segmented by spend tier.
- Foot traffic report: if physical event, measures before/after via GBP views, calls, direction requests.

## 6. Tech Stack & AI Providers

| Layer | Pick | Why |
|---|---|---|
| Frontend | Next.js on Vercel | Fast, SSR for event landing pages (SEO matters) |
| Auth | Clerk | Fastest org + user management |
| DB | Supabase (Postgres) | Auth-adjacent, realtime, generous free tier |
| Storage | Cloudflare R2 | Cheapest egress; critical when serving event images |
| Payments | Stripe (Billing + Checkout) | Subscriptions, metered billing, invoices, tax |
| Email delivery | Resend or Postmark | Deliverability for transactional + campaigns |
| Email campaigns | Customer.io or self-built on Resend | Behavior-based sequences |
| SMS | Twilio | Reminders, RSVPs |
| AI — text | Anthropic Claude API (claude-sonnet-4-6 default, opus for strategy) | Best long-form marketing copy; strong brand-voice adherence |
| AI — images | Replicate or fal.ai (Ideogram / FLUX / SDXL) | One integration, many models |
| AI — video (short) | Runway or Luma via fal.ai | Cost-controllable, good quality |
| AI — voice | ElevenLabs | Best voiceover quality |
| AI — transcription | Deepgram or AssemblyAI | Fast, cheap, good diarization |
| AI — embeddings / RAG | Voyage AI or OpenAI text-embedding-3-large + pgvector | Brand voice + event memory |
| Analytics | PostHog | Product + behavior + session replay |
| Observability | Sentry + Axiom | Errors + logs |

**Golden rule:** AI API keys live only on the backend. Every AI call is wrapped in middleware that checks user auth → subscription tier → credit balance → rate limit → then calls provider → logs cost and tokens against the user.

**IMPORTANT divergence from current codebase:** The current HypeSpace v1 is built on Express + React (Vite) + Drizzle + Postgres — NOT Next.js/Clerk/Supabase. This roadmap represents either (a) a green-field rewrite target, or (b) aspirational tech picks to re-evaluate. Any migration requires an explicit decision phase before Phase 0.

## 10. Risks & Open Questions

- AI cost inflation (video abuse) — credits + caps
- Deliverability — warm IPs, Postmark/Resend, enforce consent
- HIPAA / PHI in uploaded lists — BAA + encryption + explicit scope
- MLS licensing for realtor playbook
- Brand voice drift — RAG / fine-tune
- Review platform ToS — auto-posting rules

**Open questions to decide before Phase 1:**
1. First vertical? Realtors or Dentists (warmest network)
2. Buy vs. build email campaign engine? (start buy, migrate later)
3. Self-hosted event pages or embed on customer sites? (start self-hosted subdomain)
4. AI transparency UX: show prompt or just output? (hidden by default, toggle for power users)
